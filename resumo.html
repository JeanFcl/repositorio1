Resumo dos conteúdos

Ol e Ul

Estas duas tags são para listas, a tag "Ul" é usado para criar uma lista desordenada, em que a ordem dos elementos não importa, já o "Ol" é usado para fazer uma lista ordenada em que os elementos tem que estar em uma sequência específica e para cada item de uma lista seja ela ordenada ou não será usado a tag "li"

img

Essa teg serve para mostrar uma imagem no documento

link

Divisória

Usando a tag "div" você cria uma divisória que geralmente é usado para separa o código que se trata de uma coisa específica

label e input

As duas tags são relacionadas a preenchimento de formulário, o label seria o título e o input é a caixa de resposta tendo varias opções como: text, number e outros
